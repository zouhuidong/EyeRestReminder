## 编译环境

Visual Studio 2022 | EasyX 20220610 | [EasyWin32](https://github.com/zouhuidong/EasyWin32)

## 安装

无需安装，直接运行。如需开机自启，可以将程序的快捷方式复制到 `%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
